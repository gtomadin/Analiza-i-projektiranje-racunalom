1. DZ ARP

Projekt koji obuhvaca 1. domacu zadacu iz predmeta Analiza i projektiranje racunala pisan je u programskom jeziku java u razvojnom okruzenju Eclipse.

Za pokretanje programa potrebno je importati projekt u razvojno okruzenje. 
Eclipse tutorial: File -> Import -> General -> Existing Projects into Workspace, pritisnuti gumb Next -> U Select archive file upisati putanju do projekta, pritisnuti gumb Finish.

Implementacija razreda „Matrica“ nalazi se u packageu „matrix“ te je razdvojen na vise klasa: Matrix, MatrixOperations, MatrixReader, MatrixWriter, Substitution, Decomposition, EquationSolver.
U packageu „tasks.FirstLab“ nalaze se kodovi zadataka 1. domace zadace.
Svaka klasa sadrzi main metodu te pokretanjem tog razreda dobiva se rješenje zadatka.

U direktoriju „1. labos data“ nalaze se „.txt“ datoteke izvornih matrica korstenih u zadacima.
